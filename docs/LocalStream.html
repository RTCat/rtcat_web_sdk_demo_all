play( [containerId] [, options])

播放流,并为流生成一个播放器,播放器为<video>,可以通过 AbstractStream#getPlayer 方法获得播放器。

Parameters:
Name Type Argument Description
containerId external:String <optional>

流将会在给定的 containerId 的 div 中播放流,其中 containerId 为 css 选择器,目前支持id选择器和类选择器。
默认使用 id选择器 , 比如 containerId 为 testDiv,那么流将会在id为testDiv的div中播放流 。如果不指定containerId,默认在 id 为 local 的 div 中播放流。
options external:Object <optional>
Properties
Name Type Argument Default Description
size external:Object <optional>
 {width:300,height:300}

播放器大小
poster external:String <optional>
 实时猫logo

poster为当播放器初始化之后无视频时,播放器显示的图像,poster目前仅支持链接的形式。
